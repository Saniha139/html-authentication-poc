# HTML Authentication System POC

This project is a simple HTML-based authentication system Proof of Concept.

## Pages Included

- login.html
- register.html
- forgot-password.html
- reset-password.html
- dashboard.html

## Features

- Basic authentication flow using HTML only
- Page redirections using anchor tags
- No CSS or JavaScript used

## Navigation Flow

Login → Dashboard  
Register → Login  
Forgot Password → Login  
Reset Password → Login  
Dashboard → Logout → Login
