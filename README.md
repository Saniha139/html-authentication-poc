# HTML Authentication System POC

This project is a simple HTML-based authentication system Proof of Concept.This project is an upgraded version of my previous assignment.
Initially built using plain HTML, now enhanced using Bootstrap 5 and custom CSS.

## Features
- Login, Register, Forgot Password, Reset Password, Dashboard
- Responsive design
- Dark mode toggle
- Password strength indicator
- Form validation
- Loading spinner
- Smooth animations

## Features

- Basic authentication flow using HTML only
- Page redirections using anchor tags

## Technologies Used
- HTML
- CSS
- Bootstrap 5

## Navigation Flow

Login -> Dashboard  
Register -> Login  
Forgot Password -> Login  
Reset Password -> Login  
Dashboard -> Logout -> Login

## UI & Styling

- Bootstrap 5 integration (cards, forms, navbar, grid)
- Custom CSS styling
- Gradient background
- Google Fonts (Poppins)
- Hover effects and smooth transitions
- Box shadows and animations

## Bonus Features Implemented
- Password Strength Indicator 
- Show/Hide Password Toggle using Bootstrap Icons 
- Form Validation with Custom Error Messages 
- Loading Spinner on Button Click 
- Dark Mode Toggle 
- Animated Transitions Between Pages 

## How to Run
Open index.html in your browser
